Apk builds of [piko](https://github.com/crimera/piko) patches

# Directory Structure

```
bins/
├── apkeditor.jar          # APKEditor CLI for merging split APKs
├── cli.jar                # ReVanced CLI for patching
├── patches.rvp            # ReVanced patches file
├── big_file_merged.apk    # Merged APK (generated)
├── temp/                  # Temporary files from patching (generated)
└── result/                # Final patched APKs (generated)
```
